# Checkbox

Элемент, который позволяет ставить или снимать отметку (галочку, крестик) в поле ввода для переключения значения включено/отключено.

При необходимости ознакомьтесь с [исходным кодом](https://github.com/2e32/react-checkbox) компонента.

## Начните работать

В storybook представлены материалы, позволяющие ознакомиться с функционалом флажка.

_Свойства (props)_:

- изучите список свойств, которые принимает компонент
- изменяйте значения свойств, чтобы увидеть какой эффект они дадут

_Использование (usage)_:

- ознакомьтесь с расширенным применением галочки. Некоторые примеры могут носить практический характер
